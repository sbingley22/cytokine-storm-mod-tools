# cytokine-storm-mod-tools

Modding tools for Cytokine Storm



Download the blend file.

Paint a texture onto the character.

Save the texture and name it "Ana\_Inpatient" or another name from below to change a different costume.

Render out a preview image of the character.



Create a folder for your mod.
Put your "Ana\_Inpatient.png" and "preview.png" files in here.
Make a new document called "mod.json".

Structure the mod.json like so:
{
"costumeName": "Ana\_Inpatient",
"modName": "YOUR MOD NAME HERE"
}



If you dont want to use steam workshop you can simply just override or place the texture in the games "Cytokine Storm\_Data/StreamingAssets/" folder.



Uploading your mod to steam workshop:

You need to place the mod files into the "WorkshopMod" folder in your game directory "Cytokine Storm\_Data/WorkshopMod/". Choose browse local files in your steam options.

Now boot the game up, select Upload Mod from the main menu options, fill in the details and click upload.
