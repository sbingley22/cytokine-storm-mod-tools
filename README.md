# cytokine-storm-mod-tools
Modding tools for Cytokine Storm

Download the blend file.

Paint a texture onto the character.

Save the texture and name it "Ana_Inpatient".

Render out a preview image of the character.

Create a folder for your mod.
Put your "Ana_Inpatient.png" and "preview.png" files in here.
Make a new document called "mod.json".

Structure the mod.json like so:
{
    "costumeName": "Ana_Inpatient",
    "modName": "YOUR MOD NAME HERE"
}

Upload your mod to steam workshop.



If you dont want to use steam workshop you can simply just overide the "Ana_Inpatient.png" texture in the games "Cytokine Storm_Data/StreamingAssets/" folder.
